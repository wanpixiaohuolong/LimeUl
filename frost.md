# LimeFrost 毛玻璃效果

## 介绍

跨平台毛玻璃/模糊效果，支持 APP（Android、iOS、鸿蒙）和 H5。通过原生(UTS)或 CSS 实现。

**依赖：** 见官方文档

## 使用

```html
<l-frost :blur="10">
  <view class="content">毛玻璃效果内容</view>
</l-frost>
```

## Props 属性

| 名称 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| blur | 模糊程度(px) | number | 10 |

## Slots 插槽

default：放置在毛玻璃上的内容。

## 插件标签

- `l-frost`：组件标签
- `lime-frost`：演示标签

## 主题定制

无
