# LimeGray 全局置灰

## 介绍

整屏置灰/移除置灰的 API，适用于悼念日等场景。

**依赖：** 无

## 使用

```js
import { setGrayScreen, removeGrayScreen } from '@/uni_modules/lime-gray'

setGrayScreen()
removeGrayScreen()
```

## API

| 方法 | 说明 |
|------|------|
| setGrayScreen | 设置全局置灰 |
| removeGrayScreen | 移除置灰 |

## 插件入口

`lime-gray`：setGrayScreen、removeGrayScreen
