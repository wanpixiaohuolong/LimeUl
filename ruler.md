# LimeRuler 尺子

## 介绍

尺子组件，用于在给定范围内选择数值，常用于身高、体重等场景。属表单组件，需从插件市场导入 `lime-ruler` 使用。**付费组件。**

**依赖：** 见插件市场说明

## 使用

```html
<l-ruler v-model="value" />
```

## 说明

详细 Props、Events、Slots 请查阅 [LimeUI+ Ruler 官方文档](https://limex.qcoon.cn/components/ruler.html)。

## 插件标签

- `l-ruler`：组件标签
- `lime-ruler`：演示标签
